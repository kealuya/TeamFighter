# TeamFighter

## 功能介绍
工作场合，组内任务安排及反馈管理系统
1，可以给自己安排工作，设定优先级、deadline，并到期提醒。
2，可以给其他人安排工作，设定优先级、deadline，并可以随时提醒。
3，可以接受别人给自己安排的工作，并可以完成、详情的填写。
4，提供请假记录。提供日志记录。提供公司内部系统嵌入。
5，todo:提供代码模块集成，可以提供员工代码参照并引入开发中的系统。
